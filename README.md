# Titanic Data Analysis

## Introduction

This project performs an exploratory data analysis (EDA) on the Titanic dataset using Python.  
The notebook demonstrates how to load, clean, analyze, and visualize passenger data in order to identify survival patterns and trends among Titanic passengers.

The analysis includes handling missing values, answering important analytical questions, and creating visualizations using Matplotlib and Seaborn.

## Dataset Information

The dataset contains information about Titanic passengers, including:

- Passenger ID
- Survival status
- Passenger class
- Name
- Gender
- Age
- Number of siblings/spouses aboard
- Number of parents/children aboard
- Ticket number
- Fare
- Cabin
- Port of embarkation

### Dataset Shape
- Rows: 418
- Columns: 12

---

## Objectives

The main objectives of this project are:

1. Load and inspect the Titanic dataset
2. Clean missing or inconsistent data
3. Perform exploratory data analysis
4. Answer analytical questions such as:
   - Survival distribution
   - Gender-based survival comparison
   - Passenger class analysis
   - Age distribution
5. Create visualizations using Seaborn and Matplotlib
